# STAT 5243 Project 1: Home Credit Default Risk

This is a project assignment of Columbia University STAT GU5243.

The project includes: End-to-end data acquisition, cleaning, preprocessing, and feature engineering workflow on the Kaggle `Home Credit Default Risk` dataset, documented in `Project_1.ipynb` and reported in `PlaceHolder.pdf`.

## Project Scope

This project follows the course requirements in `Project 1.pdf`:

- Data acquisition from public data sources
- Data cleaning and inconsistency handling
- Exploratory Data Analysis (EDA)
- Preprocessing and feature engineering
- Export of cleaned modeling datasets
- Structured notebook and report deliverables

Primary artifacts:

- `Project_1.ipynb`: full pipeline implementation and analysis
- `Project 1.pdf`: project brief and deliverable requirements
- `PlaceHolder.pdf`: final report to the project findings
- `cleaned_dataset/`: final processed train/validation/test feature tables
- `log/`: feature-drop audit logs by preprocessing step

## How To Run

1. Place all required Kaggle raw files into `home-credit-default-risk/`.
2. Open `Project_1.ipynb`.
3. Run all cells from top to bottom.

The notebook:

- Builds relational aggregates across Home Credit tables
- Runs EDA and data-quality audits
- Applies missing-value handling, drift checks, winsorization, encoding, and feature pruning
- Balances the training set with `SMOTENC`
- Exports final datasets to `cleaned_dataset/`

## Pipeline Summary (from `Project_1.ipynb`)

- Initial joined data: `(356255, 894)`
- Modeling train matrix before pruning: `(246008, 891)`
- Post-cleaning (imputation + winsorization): `(246008, 698)`
- Post-pruning (low-info + high-correlation removal): `(246008, 396)`
- Final balanced train (`SMOTENC`): `(271377, 396)`
- Final validation set: `(61503, 396)`
- Final test set: `(48744, 396)`

Key outcomes:

- Total features reduced: `495`
- Synthetic samples added by SMOTENC: `25369`
- Target rate improved from `8.07%` to `16.67%` in training data
- Best grouped numeric imputation setting (validation benchmark): `OCCUPATION_TYPE + Median` with AUC `0.701886`
- Best winsorization benchmark: quantiles `0.05-0.95` with validation AUC `0.708085` (vs baseline `0.693368`)
