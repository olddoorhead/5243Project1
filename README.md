## Dataset

Source:
- https://www.kaggle.com/competitions/home-credit-default-risk/data

Download the raw files manually (Kaggle account required) and place them in `home-credit-default-risk/`.

Required files:
- `application_train.csv`
- `application_test.csv`
- `bureau.csv`
- `bureau_balance.csv`
- `previous_application.csv`
- `POS_CASH_balance.csv`
- `credit_card_balance.csv`
- `installments_payments.csv`

## Environment Setup

Use Python 3.10+.

Install notebook dependencies:

```bash
pip install numpy pandas matplotlib seaborn scipy scikit-learn imbalanced-learn jupyter
```

Optional (only needed to rebuild `Report.pdf` and `Appendix.pdf` from Markdown):

```bash
pip install markdown pypdf playwright
python -m playwright install chromium
```

## How to Run

1. Put all required Kaggle CSV files into `home-credit-default-risk/`.
2. Open and run `Project_1.ipynb` from top to bottom.
3. Verify generated outputs:
   - `cleaned_dataset/train.csv`
   - `cleaned_dataset/val.csv`
   - `cleaned_dataset/test.csv`
   - `log/feature_drop_*.csv`

Optional report rebuild command:

```bash
python scripts/export_pdf.py --root . --build-appendix-pdf
```
