# cleaned_dataset

This folder should contain the final processed datasets generated by running `Project_1.ipynb`.

## Required files

- `train.csv`: final training feature matrix after preprocessing and SMOTENC balancing
- `val.csv`: final validation feature matrix
- `test.csv`: final test feature matrix for inference/submission workflows

## Expected source

These files are created by the export step in `Project_1.ipynb` (the cell that saves outputs to `cleaned_dataset/`).

## Quick checklist

After running the notebook, this folder should have exactly:

- `train.csv`
- `val.csv`
- `test.csv`
- `README.md`

If any CSV is missing, re-run the notebook from top to bottom and confirm the final export cell completes successfully.
