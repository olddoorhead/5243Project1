# home-credit-default-risk

Place the raw Kaggle Home Credit Default Risk files in this folder.

Source:
- https://www.kaggle.com/competitions/home-credit-default-risk/data

## Required files for this project

- `application_train.csv`
- `application_test.csv`
- `bureau.csv`
- `bureau_balance.csv`
- `previous_application.csv`
- `POS_CASH_balance.csv`
- `credit_card_balance.csv`
- `installments_payments.csv`

## Optional files

- `sample_submission.csv`
- `HomeCredit_columns_description.csv`

## Quick checklist

Before running `Project_1.ipynb`, this folder should contain at least all required files above plus this README.

This folder is gitignored except `README.md`.
